Student Grade Tracker

PROJECT OVERVIEW

Student Grade Tracker is a Java console-based application developed to manage student records and analyze academic performance. 
The application allows users to add student details, view all student records, search for a student by name, and generate a performance report.

FEATURES

- Add Student Records
- View All Students
- Search Student by Name
- Generate Student Report
- Calculate Grades Automatically
- Find Average Marks
- Display Highest and Lowest Marks
- Identify the Topper
- Menu-Driven User Interface

TECHNOLOGIES USED

- Java
- ArrayList
- Scanner Class
- Object-Oriented Programming (OOP)

PROJECT STRUCTURE

StudentGradeTracker.java

HOW TO RUN THE PROJECT

Compile the Program

javac StudentGradeTracker.java

Run the Program

java StudentGradeTracker

CONCEPTS USED

- Classes and Objects
- Constructors
- Methods
- ArrayList Collection
- Loops (for / do-while)
- Conditional Statements
- User Input Handling
- Searching and Reporting

LEARNING OUTCOME

This project helped in understanding Java fundamentals, Object-Oriented Programming concepts, collections, 
loops, methods, and menu-driven application development.
